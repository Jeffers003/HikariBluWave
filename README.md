# 🌊 HikariBluWave

Uma plataforma de e-commerce moderna desenvolvida com **React**, **Vite** e **TypeScript**, focada em desempenho, design responsivo e uma experiência de compra intuitiva.

## ✨ Funcionalidades

- 🛍️ Catálogo de produtos
- 🔍 Busca de produtos
- 📂 Categorias
- ❤️ Lista de favoritos
- 🛒 Carrinho de compras
- 👤 Autenticação de usuários
- 📱 Interface totalmente responsiva
- ⚡ Alto desempenho com Vite

## 🛠️ Tecnologias

- React
- Vite
- TypeScript
- Tailwind CSS
- Chakra UI
- React Router
- ESLint

## 📁 Estrutura do Projeto

```text
src/
├── assets/

├── pages/
├── routes/

├── styles/
├── App.tsx
└── main.tsx
```

## 🚀 Como executar

### Clone o repositório

```bash
git clone https://github.com/seu-usuario/HikariBluWave.git
```

### Entre na pasta

```bash
cd HikariBluWave
```

### Instale as dependências

```bash
npm install
```

### Execute o projeto

```bash
npm run dev
```

O projeto estará disponível em:

```text
http://localhost:5173
```
